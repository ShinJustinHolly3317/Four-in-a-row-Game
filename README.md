# Game_Four-in-a-row
A advanced version of Tic-Tac-Toe
---
## How to play
1. Use left ← / right → key to move
2. Use down ↓ to drop the ball

## How to win
1. Default: 2 players
2. If there is a row of 4 balls, which is in no matter horizontal / vertical / diagonal type.

Pretty simple, get started!
